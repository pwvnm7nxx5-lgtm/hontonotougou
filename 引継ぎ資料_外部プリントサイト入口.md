## 次に実装したい内容

外部のプリントサイトへ飛べる入口ページ、または入口機能を追加したい。

元の修正案:

> 今作っているアプリから、各種プリントサイトなどに飛べるような機能をつけたい。今公開されているものの例として、スラプリやチビムスドリルなど、さまざまなサイトがある。それらのサイトの中から、プリントサイトに簡単に飛べるようにしてほしい。可能であれば、今私が作っているアプリと同じように、お気に入りの設定やブックマーク、検索などができるようにしてほしい。ただし、プリントの量がとても多いと思うため、分割して作業をする必要があるなら教えてください。

## 最近の主な実装

### 1年生 かずプリント

対象:

`apps/number-print-grade1/`

対応済み:

- 「まぜて出題」を削除。
- 「大きさくらべ」は `>、<、=` を答えさせる形式ではなく、大きい数そのものを答える形式に変更。
- 問題文は `おおきいほうをかきましょう。` に変更。
- 答えページでも大きい数が出ることをブラウザで確認済み。

### 1年生 とけいプリント

対象:

`apps/clock-print-grade1/`

対応済み:

- 「針をかく」問題は、長い針を描く問題に変更。
- 問題の時計には短い針だけが最初から描いてある。
- 答えページでは短い針と長い針の両方が出る。
- `0ふん・30ふんを時計に入れる` 設定を追加。
- 0ふん・30ふんの補助数字は時計の外側に出るようにした。
- 古い保存状態の時計SVGが残らないように `stateVersion` を上げている。

### 2年生 時刻・時間

対象:

`apps/time-print-grade2/`

対応済み:

- `5分ごとの数字をつける` 設定を追加。
- オンにすると、1の近くに5、2の近くに10、3の近くに15のように補助数字を表示する。
- 補助数字は時計の外側に出るように調整済み。
- 目盛りにかぶりにくいよう、SVGの表示範囲を広げている。
- 古い保存状態の時計SVGが残らないように `stateVersion` を上げている。

### 2年生 水のかさ

対象:

`apps/capacity-print-grade2/`

方針:

- 水のかさを読む問題などは削り、単位変換だけに絞る方針。
- やさしい問題では、上部にヒントを1か所だけ表示する。
- ヒント文:

`ヒント: 1L = 10dL、1dL = 100mL、1L = 1000mL`

## 外部プリントサイト入口の実装方針案

次の実装では、既存のトップページに直接混ぜすぎるより、まずは新しい入口カテゴリを分けた方が安全。

候補:

1. トップページに「外部教材リンク」カードを1つ追加する。
2. `apps/resource-links/` のような新アプリを作る。
3. その中で外部プリントサイトを検索・お気に入り登録できるようにする。

おすすめは 2。

理由:

- トップページが教材アプリ一覧としてすっきり保てる。
- 外部サイトの数が増えても `resource-links` 側で管理できる。
- 検索、お気に入り、タグ、カテゴリ、対象学年を独立して作れる。

## 作るとよさそうな構成

新規:

- `apps/resource-links/index.html`
- `apps/resource-links/styles.css`
- `apps/resource-links/app.js`
- `apps/resource-links/README.md`

必要なら:

- `apps/resource-links/sites.config.js`

または `app.js` 内に最初は配列で持ってもよい。

データ例:

```js
{
  id: "example-site",
  title: "サイト名",
  description: "プリント内容の説明",
  url: "https://example.com/",
  categories: ["算数", "国語"],
  grades: ["1", "2", "3"],
  tags: ["無料", "印刷", "計算"],
  note: "外部サイトへ移動します。",
}
```

## 注意点

外部サイト情報は変わる可能性があるため、実装時には現在のサイト名・URL・内容を確認すること。

外部リンクを追加するときは、次を確認する。

- 公式または実際に見えるURLか
- 教材プリントへの入口として適切か
- 無料・有料・会員登録の扱い
- 学年や教科の分類
- 著作権的に、教材そのものをコピーせずリンクに留める
