# 特別支援アプリ 開発引継ぎ資料

## 目的

このプロジェクトは、先生向けの「特別支援アプリ」専用の静的Webアプリです。
複数の教材作成アプリや外部教材検索を、トップページにまとめるハブ型アプリとして扱います。

旧・統合プロジェクトとは分けて考え、このフォルダを現在の本体として扱います。

## 基本構成

ルートはトップページです。

- `index.html`
- `styles.css`
- `app.js`
- `apps.config.js`
- `README.md`

各アプリは `apps/<app-id>/` に置きます。

- `apps/<app-id>/index.html`
- `apps/<app-id>/styles.css`
- `apps/<app-id>/app.js`
- `apps/<app-id>/README.md`

トップページは `apps.config.js` の `window.INTEGRATED_APPS` を読み、アプリカードを表示します。
各アプリには、必要に応じて `../../index.html` に戻る入口を置きます。

## トップページの現在方針

- 「特別支援アプリ」の入口として扱う。
- 学年ごとの絞り込みを使う。
- 全学年対応アプリは、1年や2年などの個別学年には入れない。
- お気に入り機能を使う。
- 外部教材はトップページのカードやカテゴリには出さない。
- 外部教材への入口は、ヘッダー右上の「外部教材を探す」ボタンだけにする。

## 外部教材検索

対象:

- `apps/resource-links/`

構成:

- `apps/resource-links/index.html`
- `apps/resource-links/styles.css`
- `apps/resource-links/app.js`
- `apps/resource-links/sites.config.js`
- `apps/resource-links/grade2-resources.config.js`

`sites.config.js` は外部教材サイト全体の入口を管理します。
`grade2-resources.config.js` は2年生向けの外部プリント候補を追加する設定ファイルです。

`apps/resource-links/index.html` では、次の順番で読み込みます。

```html
<script src="sites.config.js"></script>
<script src="grade2-resources.config.js"></script>
<script src="app.js"></script>
```

## 既存アプリの注意点

### 漢字なぞりプリント

- マス内の十字線は、文字とかぶらないよう点線にする方針。
- 薄い文字が上に来るようにする。

### 1年生 かずプリント

- 「まぜて出題」は不要。
- 大きさ比べは、1年生向けに「どちらが大きいか書きましょう」のような表現にする。
- 答えは `>` や `<` ではなく、大きい数を書く方向。
- 問題文内では、全学年で未習漢字を避け、必要に応じてひらがなにする。

### 1年生 時計

- 針を描く問題では、長い針を描かせる。
- 短い針は問題の時計に最初から描いておく。
- 補助数字は、0分と30分だけを表示するか選べるようにする。

### 2年生 時刻・時間

- 既存状態を大きく崩さない。
- 5分ごとの補助数字を表示する設定を持つ。
- 補助数字は時計の外側にはみ出す位置に置き、目盛りにできるだけ重ねない。

### 2年生 長さ

- 「単位をかえる」が「ものさしを読む」と同じ問題にならないようにする。
- `7cm 5mm = 75mm` のような単位変換問題を出す。
- 長さ比べには `どちらが長いか、>、<、=で書きましょう。` を入れる。
- `1cm 5mm + 2cm 2mm = 3cm 7mm` のようなたし算を出す。
- ひき算も出す。
- 「むずかしい」のメートルテープ形式は削除。
- 単位変換ヒント `1cm = 10mm、1m = 100cm` は「やさしい」のときだけ、プリント上部など1か所だけに表示する。

### 2年生 水のかさ

- 複雑な水量読み取りではなく、単位変換だけに絞る方針。
- やさしい問題だけ、ヒントを表示する。
- ヒントは `1L = 10dL、1dL = 100mL、1L = 1000mL`。
- 難易度ごとの差がほぼない場合は、難易度設定を削除する。

### 図形

- 問題タイプ名を書くものだけでよい。
- それ以外の余計な問題タイプは削除する。
- 四角形が斜めにずれて見える場合は、図形の座標を直接直す。
- 1年生と2年生の形プリントは、削除方針が出ているので再追加前に確認する。

## GitとCodex環境の注意

この環境では、Codexのローカルシェルから `.git` に書き込む操作が失敗することがあります。

確認済みの失敗例:

- `.git/FETCH_HEAD: Permission denied`
- `.git/index.lock: Permission denied`

そのため、fetch、switch、commit、pushなどは、ユーザー側のPowerShellで行うのが安全です。

手動で作業する場合の基本コマンド:

```powershell
cd "C:\Users\fiven\OneDrive\デスクトップ\特別支援アプリ"
git status
git fetch origin
git switch main
git pull --ff-only
```

PRブランチを確認する場合:

```powershell
cd "C:\Users\fiven\OneDrive\デスクトップ\特別支援アプリ"
git fetch origin
git switch codex/grade2-printsite
git status
```
